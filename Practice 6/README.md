# Практическая работа №6 (OpenCL)
## Тема
Программирование на OpenCL для CPU и GPU.
## Цель работы
- изучить основы OpenCL
- сделать кросс-платформенную программу для параллельных вычислений
- сравнить производительность CPU и GPU на одной задаче


Программа:
1) находит OpenCL платформу и GPU устройство  
2) создает контекст и очередь команд  
3) загружает и компилирует OpenCL ядро  
4) копирует данные на GPU  
5) запускает ядро  
6) считывает результат и проверяет корректность  
7) замеряет время CPU и GPU и выводит CSV


## Почему CPU не запускался через OpenCL
В идеале сравнение должно быть **OpenCL CPU vs OpenCL GPU**, но в **Google Colab** обычно доступен OpenCL runtime только для GPU, а **CPU OpenCL драйвер (Intel/AMD CPU runtime)** отсутствует или не подключен в окружении.

Поэтому:
- GPU тест делался **через OpenCL**
- CPU тест делался **обычным C++ кодом**, чтобы всё равно получить сравнение производительности и график

Это не ошибка кода — это ограничение окружения Colab.


## Формат вывода
Программа выводит результат в CSV:

Где:
- `n` — размер массива
- `cpu_ms` — время CPU (обычный цикл)
- `gpu_ms` — время GPU (OpenCL kernel profiling)
- `ok_cpu` — корректность CPU (1/0)
- `ok_gpu` — корректность GPU (1/0)

По этим данным был построен график **cpu vs gpu**.
