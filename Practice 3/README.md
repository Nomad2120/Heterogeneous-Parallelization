# Практическая работа №3 (CUDA)

## Тема
Реализация более сложных алгоритмов сортировки (слияние, быстрая, пирамидальная) на GPU с использованием CUDA.

## Что сделано
В этой работе я сравнил сортировки на CPU и на GPU (CUDA):
- merge sort (слияние)
- quick sort (быстрая)
- heap sort (пирамидальная)

Для каждой сортировки есть:
- последовательная версия на CPU
- параллельная версия на GPU (CUDA)

Все задания реализованы в **одном файле** `main.cu`.

## Среда
Код запускался в **Google Colab** с поддержкой CUDA (GPU).

## Проверка размеров
Тестирование делается на массивах:
- 10 000
- 100 000
- 1 000 000

Программа выводит время выполнения для CPU и GPU и проверяет, что массив после сортировки реально отсортирован.
