# Assignment 2  
OpenMP, CUDA и гетерогенные вычисления

## Описание
В этом задании рассматриваются основы гетерогенной параллелизации, а также практическая работа с параллельными вычислениями на CPU (OpenMP) и GPU (CUDA).

## Среда выполнения
Для выполнения всех заданий использовался **Google Colab** с поддержкой:
- OpenMP для CPU
- CUDA для GPU

## Вывод
Assignment 2 позволил:
- разобраться в принципах гетерогенной параллелизации;
- получить практический опыт работы с OpenMP и CUDA;
- увидеть разницу в производительности между CPU и GPU;
- понять, в каких задачах параллельный и гетерогенный подход даёт наибольший эффект.
